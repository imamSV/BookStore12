# BookStore Engine (Flask)
функциональное веб-приложение для книжного интернет-магазина.
Проект демонстрирует навыки построения масштабируемых CRUD-систем, работы с реляционными базами данных и реализации бизнес-логики на стороне сервера.

---

## Основные возможности
* **User System:** Полный цикл регистрации и аутентификации пользователей с использованием сессий (Flask-Login).
* **Security:** Безопасное хранение паролей с использованием современных алгоритмов хеширования (Werkzeug).
* **Product Management:** Динамический каталог книг с интеграцией через ORM.
* **Shopping Cart:** Реализация логики корзины (добавление, удаление позиций, динамический пересчет суммы).
* **Orders & History:** Система оформления заказов с сохранением транзакционной истории в БД.
* **Social Features:** Возможность оставлять отзывы и выставлять рейтинги товарам.
* **Migrations:** Управление версиями схемы базы данных (Alembic).

---
## REST API

В проекте реализован дополнительный API-слой для работы с данными в формате JSON.
Доступные endpoints:
* **GET /api/books — получение списка книг
* **GET /api/books/<id> — получение информации о конкретной книге
* **GET /api/cart — получение корзины текущего пользователя
* **GET /api/orders — получение списка заказов пользователя
* **API может использоваться для подключения внешнего frontend-приложения или мобильного клиента.
---
## Технологический стек
* **Core:** Python 3.10+, Flask 3.1
* **Database:** SQLAlchemy 2.0 (ORM), SQLite
* **Migrations:** Flask-Migrate (Alembic)
* **Security:** Flask-Login, Werkzeug
* **Configuration:** python-dotenv
* **Frontend:** Jinja2, HTML5, CSS3

---

## ⚙️ Установка и запуск

1. **Окружение:** `python -m venv venv` и активировать его.
2. **Библиотеки:** `pip install -r requirements.txt`
3. **База данных:** `flask db upgrade`
4. **Старт:** `python run.py`
