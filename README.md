# BookStore Flask
Интернет-магазин книг на Flask.

## Функции
- регистрация пользователей
- каталог книг
- корзина
- оформление заказа
- история заказов
- отзывы и рейтинг

## Технологии
Python  
Flask  
SQLAlchemy  
SQLite  

## Установка
pip install -r requirements.txt

## Запуск
```python run.py```
